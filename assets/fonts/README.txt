Place a TrueType font here named font.ttf for the cleanest setup.

If this directory is empty the program automatically falls back to
common system fonts (DejaVuSans on Linux, Arial on Windows/macOS).

Recommended quick setup on Debian / Ubuntu:

    sudo apt install fonts-dejavu-core
    cp /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf assets/fonts/font.ttf

On Fedora:

    sudo dnf install dejavu-sans-fonts
    cp /usr/share/fonts/dejavu/DejaVuSans.ttf assets/fonts/font.ttf
